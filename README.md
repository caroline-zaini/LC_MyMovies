# To run the app localy :

## the backend  (expressJS) :

  npm install

  npm install sync-request --save

  npm install mongoose --save
  
  npm start

  http://localhost:3000
  
## the frontend (reactJS)

  npm install --global create-react-app
  
  npm start
 
  then choose another port
